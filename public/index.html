Discord Open Tools

Integration tools for connecting Discord with GitHub and Trello

GitHub Integration

Receive notifications in Discord when activity happens in your GitHub repositories.

Webhook URL: /githubNotification

Trello Integration

Get Discord notifications when cards are created, moved, or commented on in Trello.

Webhook URL: /trelloNotification

Register Trello Board

Connect your Trello board to Discord by visiting:

/registerTrelloBoard?boardId=YOUR_BOARD_ID

Replace YOUR_BOARD_ID with the ID of your Trello board.
